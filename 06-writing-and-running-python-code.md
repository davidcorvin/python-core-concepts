# Writing and Running Python Code

Python code can be written and executed using various methods. Here are five popular ways to write and run Python code along with their setup instructions, advantages, and disadvantages:

## 1. IDLE
### Setup Instructions:
- Install Python from [python.org](https://www.python.org).
- IDLE comes bundled with Python installations. Launch IDLE from your applications.

### Advantages:
- Simple and easy to use for beginners.
- Built-in Python Shell for immediate feedback.

### Disadvantages:
- Limited features compared to other IDEs.
- Can be slow for larger projects.

## 2. VS Code
### Setup Instructions:
- Download and install [Visual Studio Code](https://code.visualstudio.com/).
- Install the Python extension from the Extensions Marketplace.

### Advantages:
- Highly customizable with extensions.
- Integrated terminal and version control.

### Disadvantages:
- Can be overwhelming for beginners due to numerous features.
- Requires setup for optimal experience.

## 3. Terminal/Command Prompt
### Setup Instructions:
- Ensure Python is installed and added to your system's PATH.
- Open your terminal or command prompt and type `python` to start.

### Advantages:
- Lightweight and fast.
- No additional software required.

### Disadvantages:
- Not user-friendly for beginners.
- Limited debugging and editing features.

## 4. Online Compilers
### Setup Instructions:
- Search for online Python compilers (e.g. [Replit](https://replit.com/), [Google Colab](https://colab.research.google.com/)).

### Advantages:
- No installation required.
- Easily shareable code snippets.

### Disadvantages:
- Dependent on internet access.
- Limited resources compared to local execution.

## 5. Jupyter Notebook
### Setup Instructions:
- Install Anaconda distribution or run `pip install notebook`.
- Launch Jupyter Notebook by typing `jupyter notebook` in your terminal.

### Advantages:
- Interactive environment ideal for data analysis.
- Supports inline visualization.

### Disadvantages:
- Can be resource-intensive.
- Steeper learning curve for beginners.
